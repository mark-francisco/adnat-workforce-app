# README

## Requirements:

```
Ruby 3.0.0 and Rails 6.1.3
```

## Instructions:

### Install Ruby gems:

```
bundle install
```

### Start server:

```
rails s
```

### Navigate to Rails app in browser. Go to this URL:

```
http://localhost:3000/
```

### Navigate to Vue app in browser. Go to this URL:

```
http://localhost:8080/
```

### Log in:

```
email: test@example.com
password: password
```
